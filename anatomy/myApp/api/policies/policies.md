# myApp/api/policies
### 目的
ここは`policy`ファイルを格納するフォルダです。ポリシーファイルはアプリケーションの中のコントローラアクションに対するアクセス認可をexpressのミドルウエアが行うために必須の.jsファイルです。
 
もし、`http://prism.gov/rick1983`には`rick1983`というユーザのみがアクセスできるようにしたいときは、このファイルがそのロジックを置く場所です。
