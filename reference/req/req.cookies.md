# req.cookies
リクエスト(`req`)に含まれる全ての[**サインされてないクッキー**](https://github.com/balderdashy/sails-docs/blob/master/PAGE_NEEDED.md) を含みます。


### 使い方
```javascript
req.cookies;
```


### 例
"chocolatechip"と名付けられ、"Yummyの値を持つクッキーが含まれると仮定します。:

```javascript
req.cookies.chocolatechip;
// "Yummy"
```
