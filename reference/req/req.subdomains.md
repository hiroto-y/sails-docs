# req.subdomains
リクエストのURLにある全てのサブドメインの配列

### 使い方
```javascript
req.subdomains;
```

### 例

リクエストされたURLが"https://ww3.staging.ibm.com"の場合:

```javascript
req.subdomains;
// -> ['ww3', 'staging']
```
